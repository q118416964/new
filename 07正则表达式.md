### 具有特殊意义的元字符
+ \d -> 匹配一个0-9的数字,相当于[0-9],和它相反的是\D ->匹配一个除了0-9的任意字符
+ \w -> 匹配一个0-9、a-z、A-Z、_的数字或字符,相当于[0-9a-zA-Z_]
+ \s -> 匹配一个空白字符(空格、制表符...)
+ \b -> 匹配一个单词的边界
+ \t -> 匹配一个制表符
+ \n -> 匹配一个换行
+ . -> 匹配一个除了\n以外的任意字符
+ ^ -> 以某一个元字符开头
+ $ -> 以某一个元字符结尾
+ \ -> 转义字符
+ x|y -> x或者y的一个
+ [xyz] -> x、y、z中的任意一个
+ [^xyz] -> 除了xyz中的任意一个字符
+ [a-z] -> 匹配a-z中的任意一个字符
+ [^a-z] -> 匹配除了a-z中的任意一个字符
+ () -> 正则中的分+ 
### 代表出现次数的"量词元字符"
+ ->+ : 出现一到多次
+ ->* : 出现零到多次
+ ->? : 出现零到一次
+ ->{n} : 出现n次
+ ->{n,} : 出现n到多次
+ ->{n,m} : 出现n-m次
